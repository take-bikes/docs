## 第63回 千葉大祭 準備手順ガイド

千葉大学サイクリング部向け｜企画準備と手続きの流れ

## 準備スケジュール

| 月 | 主なタスク |
|----|--------------|
| 5月 | 参加登録フォーム回答 / 企画案検討・決定 / 責任者・副責任者の決定 |
| 6月 | 企画概案調査フォーム回答 / 誓約書・企画書・食材リスト提出 |
| 7月 | 第1回企画説明会 / 危険物・レンタル物品申請 |
| 8月 | 不備書類の再提出 / レンタル物品説明会参加 |
| 10月 | 最終説明会参加 / 検便・講習関係完了 |

## 各月の詳細手順

<Tabs>
<Tab title="5月：参加登録・企画決定">
<AccordionGroup>
  <Accordion title="企画担当者の決定">
    部会で企画担当者を決定。テント企画・自由企画ともに2年生2名ずつが担当。
  </Accordion>
  
  <Accordion title="参加登録">
    実行委員会配布のGoogleフォームに回答し、団体情報・連絡先・企画概要を入力。
  </Accordion>
  
  <Accordion title="企画案の決定">
    前年の事例を参考に、部会で企画案を検討し決定。不決の場合は責任者判断。
  </Accordion>
</AccordionGroup>
</Tab>

<Tab title="6月：書類提出">
<AccordionGroup>
  <Accordion title="企画概案調査">
    6月15日までにGoogleフォームで回答。内容：機材・物品、安全対策、収支見込みなど。
  </Accordion>
  
  <Accordion title="誓約書提出">
    顧問・企画責任者の署名捺印必須。PDF形式で事務局に提出。
  </Accordion>
  
  <Accordion title="企画書・食材・参加者リスト">
    7月末までにすべての書類を提出。アレルゲン情報は必ず明記。
  </Accordion>
</AccordionGroup>
</Tab>

<Tab title="7月：物品申請">
<AccordionGroup>
  <Accordion title="危険物・発電機使用申請">
    使用申請書、誓約書、安全対策書が必要。講習受講必須。
  </Accordion>
  
  <Accordion title="レンタル物品申請">
    Googleフォームでテーブル・椅子・テントなどを申請（先着順）。
  </Accordion>
  
  <Accordion title="第1回企画説明会">
    全体スケジュールの確認、書類提出の注意事項、質疑応答を行う。
  </Accordion>
</AccordionGroup>
</Tab>

<Tab title="8月：再提出・確認">
<Warning>
  不備があった書類の修正版を8/15までに提出。追加資料の確認も行う。
</Warning>

<AccordionGroup>
  <Accordion title="書類再提出">
    指摘事項を修正し、追加書類も提出する。
  </Accordion>
  
  <Accordion title="レンタル物品説明会">
    使用方法の実演、注意事項の確認、責任分担の明確化を行う。
  </Accordion>
</AccordionGroup>
</Tab>

<Tab title="10月：最終準備">
<AccordionGroup>
  <Accordion title="衛生・防火講習会">
    食品衛生、防火対策、消火実習を含む。全員参加が必須。
  </Accordion>
  
  <Accordion title="最終説明会・検便">
    各種説明会への参加、検便提出、健康管理確認を実施。
  </Accordion>
  
  <Accordion title="集金・検便回収">
    参加費の集金、検便キット配布・回収、健康管理の確認を行う。
  </Accordion>
</AccordionGroup>
</Tab>
</Tabs>
