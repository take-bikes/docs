# 🚨 緊急時対応

<Warning>
  このページでは、緊急事態が発生した際の対応手順を解説します。
  全スタッフが内容を理解し、冷静に対応できるよう準備してください。
</Warning>

---

## 雨天時の判断基準

<Info>
  天候による運営判断は、部長が最終決定します。
  以下の基準を参考に、早めに判断してください。
</Info>

### 判断フロー

```
         ┌─────────────────┐
         │ 天気予報確認     │
         │ （前日夜・当日朝）│
         └────────┬────────┘
                  ▼
         ┌─────────────────┐
         │ 降水確率は？      │
         └────────┬────────┘
                  ▼
    ┌─────────────┼─────────────┐
    ▼             ▼             ▼
  30%以下      30-70%        70%以上
    │             │             │
    ▼             ▼             ▼
  通常運営     準備して      中止の
               待機         可能性高
```

### 具体的な判断基準

| 条件 | 飲食部門 | 輪タク部門 |
|------|----------|------------|
| 晴れ・曇り | 通常運営 | 通常運営 |
| 小雨（一時的） | 通常運営 | **一時中断** |
| 雨（継続的） | テント内で継続 | **運行中止** |
| 強風（5m/s以上） | テント補強 | **運行中止** |
| 雷 | **即時中止** | **即時中止** |

<Warning>
  **輪タクは路面が濡れたら即座に運行中止**としてください。
  スリップによる転倒リスクが非常に高まります。
</Warning>

### 中止決定の連絡フロー

<Steps>
  <Step title="部長が判断">
    天候・安全状況を確認し、中止を決定する。
  </Step>
  
  <Step title="各責任者に連絡">
    飲食責任者・輪タク責任者に電話で連絡する。
  </Step>
  
  <Step title="スタッフへ周知">
    LINEグループで全スタッフに周知する。
  </Step>
  
  <Step title="来場者への告知">
    テント前に「本日中止」の掲示を出す。
  </Step>
</Steps>

## 怪我人発生時の対応

<Warning>
  怪我人が発生した場合は、以下のフローに従って対応してください。
  **人命最優先**です。迷ったら救急車を呼んでください。
</Warning>

### 対応フロー

```
       ┌─────────────────┐
       │ 怪我人発生！     │
       └────────┬────────┘
                ▼
       ┌─────────────────┐
       │ 周囲の安全確保   │
       │ （二次事故防止） │
       └────────┬────────┘
                ▼
       ┌─────────────────┐
       │ 意識はあるか？   │
       └────────┬────────┘
                ▼
    ┌───────────┴───────────┐
    ▼                       ▼
  意識なし                意識あり
    │                       │
    ▼                       ▼
  即座に119番         ┌─────────────┐
  AED準備            │ 怪我の程度は？│
                     └──────┬──────┘
                            ▼
              ┌─────────────┼─────────────┐
              ▼             ▼             ▼
            重傷          中程度         軽傷
            （出血多い、   （打撲、      （擦り傷
             動けない）    捻挫など）    程度）
              │             │             │
              ▼             ▼             ▼
           119番        保健室へ      応急処置
            連絡         付き添い      して様子見
```

### 応急処置キットの内容

<Check>
  以下のアイテムがテント内の応急処置キットに含まれているか確認してください。
</Check>

| アイテム | 数量 | 用途 |
|----------|------|------|
| 絆創膏（各サイズ） | 20枚 | 切り傷・擦り傷 |
| 消毒液 | 1本 | 傷口の消毒 |
| ガーゼ | 10枚 | 止血・保護 |
| 包帯 | 2巻 | 固定・止血 |
| 冷却スプレー | 1本 | 打撲・捻挫 |
| 保冷剤 | 3個 | 冷却 |

### 救急車を呼ぶ際の伝達事項

<Steps>
  <Step title="119番に電話">
    「救急です」と伝える。
  </Step>
  
  <Step title="場所を伝える">
    「〇〇大学の大学祭会場、〇〇テント前です」
  </Step>
  
  <Step title="状況を伝える">
    「〇歳くらいの方が〇〇（症状）です」
  </Step>
  
  <Step title="指示を受ける">
    消防からの指示に従う。電話を切らずに待機。
  </Step>
</Steps>

## その他のトラブル対応

### 機材トラブル

| トラブル | 対応 |
|----------|------|
| ホットサンドメーカー故障 | 予備に切り替え。予備もなければ販売一時停止 |
| 電源トラブル | 大学祭本部に連絡し、電気担当を呼ぶ |
| 輪タク車両トラブル | 即時運行中止、安全な場所へ移動 |

### クレーム対応

<Steps>
  <Step title="まず謝罪">
    「ご不便をおかけして申し訳ございません」
  </Step>
  
  <Step title="話を聞く">
    お客様の話を最後まで聞く。遮らない。
  </Step>
  
  <Step title="責任者を呼ぶ">
    対応が難しい場合は、責任者に引き継ぐ。
  </Step>
</Steps>
