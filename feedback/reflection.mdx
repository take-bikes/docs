# 🗣️ 反省・申し送り事項

<Info>
  プロジェクト終了後は必ず「振り返り（Reflection）」を行いましょう。
  ここで挙がった課題と改善策が、来年の成功の鍵となります。
</Info>

---

## KPT振り返り（Focus Review）

今年度の運営終了後に行った振り返りミーティングの結果です。

### Keep（良かったこと・継続したいこと）

<Check>
  - [x] ホットサンドの「チョコバナナ」が予想以上に売れた（完売）
  - [x] 輪タクのオンライン予約システム（Googleフォーム）が好評だった
  - [x] シフトの休憩時間を30分確保したことで、スタッフの疲労が軽減された
  - [x] 声出し呼び込みの効果が高かった
</Check>

### Problem（悪かったこと・課題）

<Warning>
  - [ ] 初日の午前中、ブレーカーが落ちてホットサンドメーカーが止まった（電圧計算ミス）
  - [ ] 輪タクの待ち時間が長くなり、一部のお客様を帰してしまった
  - [ ] お釣りの100円玉が不足し、両替に走ることになった
  - [ ] ゴミ箱がすぐに溢れてしまった
</Warning>

### Try（次はこうしたい・改善案）

<Steps>
  <Step title="電源管理の徹底">
    使用する電気機器の総ワット数を計算し、発電機の容量または使用コンセントを分散させる。
  </Step>
  <Step title="整理券システムの導入">
    輪タクは「〇〇時から」の整理券を配布し、待ち時間を自由に過ごしてもらう。
  </Step>
  <Step title="釣銭準備金の増額">
    100円玉は50枚ではなく、100枚以上用意する。
  </Step>
  <Step title="ゴミ回収頻度の向上">
    1時間ごとの巡回シフトに「ゴミ箱チェック」を組み込む。
  </Step>
</Steps>

## 次年度への申し送り（重要）

<Warning>
  **これだけは伝えておきたい！**
</Warning>

1.  **保健所の申請書類**は、書式が変わることがあるので毎年最新版を確認すること。
2.  **学祭実行委員との連絡**は、レスポンスを早くすること（場所取りに影響します）。
3.  **輪タクのタイヤ**は、大祭前に必ず新品に交換すること（パンクリスク低減）。
4.  **雨天時の代替案**（レクリエーションなど）を事前に考えておくと、中止時の士気低下を防げる。

<Note>
  このドキュメントを読んだ後輩たちへ：
  完璧な運営はありません。自分たちらしさを大切に、楽しみながら運営してください！
</Note>
