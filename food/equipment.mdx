## 必須設備（要項指定）

<Warning>
  これらが欠けていると、当日検査で**不合格**となり営業できません。
</Warning>

- [ ] **テント**: 2.7m × 3.6m（1.5間×2間）の4脚テント
- [ ] **遮熱板**: 60cm × 90cm以上の不燃材（コンロ周り用）
- [ ] **ビニールシート**: 長机を覆うサイズ（衛生確保用）
- [ ] **10型消火器**: 本部レンタル品（当日受取）
- [ ] **蓋つきゴミ箱**: 密閉できるもの（45L以上推奨）
- [ ] **温度計**: クーラーボックス用（食材管理）
- [ ] **消毒液**: アルコール製剤（手指用・器具用）

---

## 調理機材（ホットサンド・コーヒー）

- [ ] **ホットサンドメーカー**: 電気式 3台（予備1台）
- [ ] **LPGボンベ**: プロパンガス（契約業者より搬入）
- [ ] **ガスコンロ/鋳物コンロ**: 安定した台の上に設置
- [ ] **チェーン**: ボンベ固定用（南京錠付き推奨）
- [ ] **電気ケトル/ポット**: コーヒー用お湯確保
- [ ] **ドリッパー・サーバー**: コーヒー抽出用
- [ ] **キッチンタイマー**: 焼き時間管理

## 消耗品・備品

- [ ] **長机**: 3台（調理用2台、販売用1台）

- [ ] **養生テープ**: 配線固定、掲示物用
- [ ] **電源コード（ドラム）**: 防雨型、十分な長さ
- [ ] **給水タンク**: 20L（蛇口付き）
- [ ] **排水バケツ**: 20L以上
- [ ] **洗剤・スポンジ**: 片付け用
- [ ] **ペーパータオル**: 手拭き用
