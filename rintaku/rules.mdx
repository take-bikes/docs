# 🚜 自由企画レギュレーション

<Info>
  輪タクは大学祭において「自由企画」という区分に分類されます。
  テント企画とは異なるルール適用となるため、以下の規定を遵守してください。
</Info>

## 1. 企画概要・定義

<CardGroup cols={2}>
  <Card title="自由企画の定義" icon="book">
    テント・部屋・ステージ・ストリートパフォーマンス企画の**いずれにも該当しない**参加形態。
  </Card>
  <Card title="具体例" icon="lightbulb">
    - 部屋企画外の教室利用
    - 屋外パフォーマンス
    - **移動式企画（輪タク）**
    - 物品販売
  </Card>
</CardGroup>

## 2. 参加条件・手続きの流れ

### 必須事項
- **説明会出席**: 全3回の説明会に**すべて出席**すること。
- **調整会議**: 「かたらいの森」等の共用スペースを使用する場合は参加必須。輪タクの場合は参加しなくてよい。

## 3. 施設利用・交渉

<Warning>
  輪タクの乗車場（スタート地点）以外に、車両を停めておく**待機場所**が必要な場合は、
  必ず「施設利用申請書」を提出してください（提出がないと確保できません）。
</Warning>

### 施設交渉の基本
- **学生支援課管轄施設**: 必要な説明会出席・書類提出を行えば、大祭事務局が交渉を代行します。
- **管轄外施設**: 各自で交渉を行い、その結果をもとに企画書を提出します。
- **体育施設**: 原則利用不可。

### 実施場所の制限・可否
<Accordion title="場所ごとの使用可否詳細">
  | 場所 | 使用可否・条件 |
  | :--- | :--- |
  | **大学会館** | 事務局使用部屋以外は使用可。 |
  | **かたらいの森** | 使用可（テント通りに面する部分は不可）。7月に調整会議あり。 |
  | **けやき会館** | **使用不可** |
  | **大学構内の食堂** | 例年使用している団体以外は使用不可。 |
  | **工学部・理学部棟** | 各学部祭に参加する団体以外は使用不可。 |
  | **教育学部棟** | 原則使用不可。 |
  | **文・法政経学部棟** | 原則使用不可。 |
  | **ライフセンター・図書館間** | **人通りが多く危険なため使用不可**。 |
  
</Accordion>

### 輪タクの走行エリアについて
- **テント通り**: ❌ 原則走行不可（人混みのため）
- **外周道路**: ⭕️ 推奨ルート（正門～南門のルート）

## 4. 禁止事項

### 🚫 自由企画特有の禁止事項
<AccordionGroup>
  <Accordion title="食品の売り歩き（一部禁止）" icon="ban">
    - テントや部屋等で調理された食品の売り歩き。
    - 密閉されていない食品の売り歩き（衛生上の問題のため）。
    
    <Check>
      **輪タクでの対応**
      - 輪タクの荷台にホットサンドを積み、販売しながら走行するのは**NG**です。
      - テントで購入したお客様が、食べながら乗車するのは**OK**です。
    </Check>
  </Accordion>
  
  <Accordion title="服装・配布物" icon="shirt">
    - **服装**: 顔が見えない着ぐるみの着用は禁止。
    - **配布**: 企画の宣伝と関係ないものの配布は禁止。
  </Accordion>

  <Accordion title="共通禁止事項" icon="triangle-exclamation">
    - 事前に自由企画として申請のないものの売り歩き。
    - 大学構内での飲酒、喫煙。
    - 企業や政治・宗教団体の宣伝活動。
  </Accordion>
</AccordionGroup>

## 5. 申請不要な活動
以下の活動は「自由企画」としての申請は不要ですが、場合により学生支援課への申請が必要です。
- 企画宣伝目的のビラ配布（構内のみ）。
- ボード等を使用して行う宣伝。
- チケットの売り歩き。
- 募金活動（要申請）。
