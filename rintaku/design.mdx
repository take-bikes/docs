## 製作プロセス

<Steps>
  <Step title="構造設計 (CAD Design)">
    3D CADソフトを使用してモデリングを行います。
    応力解析により、負荷のかかるポイントを特定し、補強リブの配置を決定します。
  </Step>

  <Step title="材料選定・調達">
    - **フレーム**: A6063アルミ角パイプ（軽量かつ高剛性）
    - **連結部**: S45C炭素鋼（耐久性重視）
    - **床板**: マリン合板（耐水性・強度）
  </Step>

  <Step title="フレーム溶接 (Welding)">
    専用の治具（ジグ）を用いて歪みを抑えながらTIG溶接を行います。
    特に結合部は二重ビードで強度を確保しています。
  </Step>

  <Step title="組み上げ・塗装 (Assembly)">
    錆止め塗装と仕上げ塗装を行い、木製パーツとホイールを組み付けます。
    最後に安全装備（反射板、ライト）を取り付けて完成です。
  </Step>
</Steps>

## リヤカーの特徴

### 連結機構

<Warning>
  連結部は走行中に最も負荷がかかる部分です。
  毎回の運行前に必ず点検してください。
</Warning>

#### 連結方式

- **ヒッチタイプ**: ボール式ヒッチカプラー
- **取り付け位置**: 自転車リアアクスル
- **安全装置**: 脱落防止ピン + セーフティチェーン

#### 連結手順

<Steps>
  <Step title="自転車を安定させる">
    自転車を平らな場所に置き、スタンドで安定させる。
  </Step>
  
  <Step title="ヒッチを接続する">
    客車のカプラーをヒッチボールに嵌める。
    「カチッ」と音がするまで押し込む。
  </Step>
  
  <Step title="ピンで固定する">
    脱落防止ピンを挿入し、ロックする。
  </Step>
  
  <Step title="セーフティチェーンを接続">
    万が一ヒッチが外れた場合の保険として、セーフティチェーンを自転車フレームに接続する。
  </Step>
  
  <Step title="動作確認">
    左右に軽く揺らし、ガタつきがないか確認する。
  </Step>
</Steps>

### タイヤ

### ブレーキシステム

リヤカーにブレーキは搭載せず、牽引車のブレーキのみで制御します。
そのため、連結部分に負荷がかかりやすいです。

## 整備基準

### 日次点検（運行前）

<Check>
  毎日の運行開始前に、以下の項目を点検してください。
  点検結果は記録簿に記入すること。
</Check>

| 点検項目 | 確認内容 | ✓ |
|----------|----------|---|
| タイヤ空気圧 | 規定値（前後とも） | |
| ブレーキ効き | 十分な制動力 | |
| 連結部 | ガタつき・緩みなし | |
| チェーン | たるみ・錆なし | |
| ライト・反射板 | 点灯・視認性確認 | |
| シートベルト | 破損・ロック確認 | |

### 週次整備

| 整備項目 | 内容 |
|----------|------|
| チェーン注油 | 専用オイルで注油 |
| ボルト増し締め | 各部のボルトを確認 |
| 清掃 | 車体全体の洗浄 |

### 月次整備

| 整備項目 | 内容 |
|----------|------|
| ブレーキシュー点検 | 摩耗状態の確認、必要なら交換 |
| タイヤ点検 | 溝の深さ、ひび割れ確認 |
| ベアリング点検 | 異音・ガタの確認 |

<Warning>
  整備に不安がある場合は、必ず経験者または専門店に相談してください。
</Warning>
