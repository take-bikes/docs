## 製作プロセス

<Steps>
  <Step title="構造設計 (CAD Design)">
    3D CADソフトを使用してモデリングを行います。
    応力解析により、負荷のかかるポイントを特定し、補強リブの配置を決定します。
  </Step>

  <Step title="材料選定・調達">
    - **フレーム**: A6063アルミ角パイプ（軽量かつ高剛性）
    - **連結部**: S45C炭素鋼（耐久性重視）
    - **床板**: マリン合板（耐水性・強度）
  </Step>

  <Step title="フレーム溶接 (Welding)">
    専用の治具（ジグ）を用いて歪みを抑えながらTIG溶接を行います。
    特に結合部は二重ビードで強度を確保しています。
  </Step>

  <Step title="組み上げ・塗装 (Assembly)">
    錆止め塗装と仕上げ塗装を行い、木製パーツとホイールを組み付けます。
    最後に安全装備（反射板、ライト）を取り付けて完成です。
  </Step>
</Steps>

## リヤカーの特徴

<AccordionGroup>
  <Accordion title="連結機構" icon="link" defaultOpen={true}>
    <Warning>
      連結部は走行中に最も負荷がかかる部分です。**毎回の運行前に必ず点検**してください。
    </Warning>

    <Card title="連結スペック" icon="gears">
      - **ヒッチタイプ**: ボール式ヒッチカプラー
      - **取り付け位置**: 自転車リアアクスル
      - **安全装置**: 脱落防止ピン + セーフティチェーン
    </Card>

  </Accordion>

  <Accordion title="車体">

    | 項目 | 仕様 |
    | :--- | :--- |
    | 全長 | 2200mm |
    | 全幅 | 1230mm |
    | 全高 | 850mm |
    | 床面高 | 350mm |
    | 最大積載重量 | 350kg |
    | 車輪 | 26インチ |

    <Card title="リヤカー基礎図面 (PDF)" icon="file-pdf" href="https://drive.google.com/file/d/1AJK20LYWfVkGv-nL4iu4qT9QYhQdyqgw/view?usp=sharing">

  <Accordion title="ブレーキ" icon="truck-ramp-box">
    <CardGroup cols={2}>
      <Card title="ブレーキシステム" icon="brake-warning">
        リヤカー本体にブレーキは搭載されていません。

      </Card>

    </Info>

## 点検

<Check>
  運行開始前に、以下の項目を点検してください。
</Check>

| 点検項目 | 確認内容 |
|----------|----------|
| タイヤ空気圧 | 規定値（前後とも） |
| ブレーキ効き | 十分な制動力 |
| 連結部 | ガタつき・緩みなし |
| チェーン | たるみ・錆なし |
| 装飾 | 破損・脱落なし |
