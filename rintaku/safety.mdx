# ⚠️ 輪タク安全基準

<Warning>
  このページは輪タク運行の**最重要ドキュメント**です。
  すべての運転手はこの内容を熟読し、遵守してください。
  違反は重大事故につながる可能性があります。
</Warning>

---

## 禁止事項・注意事項

<Warning>
  **絶対禁止事項**
  
  以下の行為は重大事故の原因となります。絶対に行わないでください。
</Warning>

- **下り坂での加速禁止**: 下り坂では必ず**徐行**（時速5km以下）し、常にブレーキに手をかけておくこと
- **定員オーバー禁止**: 乗客は**最大2名**まで。子供であっても3名以上は絶対不可
- **シートベルト装着確認**: 乗客のシートベルトを**必ず確認してから発車**すること
- **片手運転禁止**: 両手でハンドルを握り、安定した姿勢を維持すること
- **イヤホン・ヘッドホン禁止**: 周囲の音が聞こえない状態での運転は禁止
- **飲酒後の運転禁止**: 前日の飲酒でも、体調に不安があれば運転しないこと

<Warning>
  **悪天候時の運行禁止**
  
  以下の条件では運行を**中止**してください。
</Warning>

- 雨天（小雨でも路面が濡れている場合）
- 強風（風速5m/s以上）
- 視界不良（霧、夕暮れ以降）
- 路面凍結の恐れがある場合

## 始業前点検チェックリスト

<Check>
  毎回の運行開始前に、以下の項目を**必ず**点検してください。
  1つでも問題があれば、修理完了まで運行しないこと。
</Check>

### ブレーキ

<Check>
  **前輪ブレーキ**
  - [ ] レバーを握って、リムにシューが正しく当たるか
  - [ ] 効きは十分か（車輪を回して確認）
  - [ ] ワイヤーに錆・ほつれがないか
</Check>

<Check>
  **後輪ブレーキ**
  - [ ] レバーを握って、リムにシューが正しく当たるか
  - [ ] 効きは十分か
  - [ ] ワイヤーに錆・ほつれがないか
</Check>

### タイヤ空気圧

<Check>
  **前輪・後輪タイヤ**
  - [ ] 空気圧は規定値か（タイヤ側面に記載、通常60-80psi）
  - [ ] ひび割れ・異物刺さりはないか
  - [ ] 溝の深さは十分か
</Check>

<Check>
  **客車タイヤ**
  - [ ] 空気圧は規定値か
  - [ ] 左右均等に入っているか
</Check>

### 連結部

<Check>
  **ヒッチ・カプラー**
  - [ ] ヒッチボールにカプラーが正しく嵌っているか
  - [ ] 脱落防止ピンが挿入・ロックされているか
  - [ ] セーフティチェーンが接続されているか
  - [ ] ガタつき・緩みはないか（左右に揺らして確認）
</Check>

### その他

<Check>
  **シートベルト**
  - [ ] ベルトに破損・ほつれがないか
  - [ ] バックルが正常にロックするか
</Check>

<Check>
  **ライト・反射板**
  - [ ] 前照灯が点灯するか
  - [ ] 尾灯（客車後部）が点灯するか
  - [ ] 反射板が汚れていないか
</Check>

<Check>
  **ベル**
  - [ ] ベルが正常に鳴るか
</Check>

## 乗客への安全説明

<Steps>
  <Step title="乗車前の説明">
    「これから輪タクにご乗車いただきます。安全のため、いくつかお願いがあります。」
  </Step>
  
  <Step title="シートベルト装着">
    「シートベルトを必ず装着してください。お子様も同様です。」
    運転手が装着を確認すること。
  </Step>
  
  <Step title="乗車中の注意">
    「走行中は立ち上がったり、手を外に出したりしないでください。」
  </Step>
  
  <Step title="緊急時の対応">
    「何かあればすぐに声をかけてください。停車します。」
  </Step>
</Steps>

## 事故発生時の対応

<Warning>
  事故が発生した場合は、以下の手順で対応してください。
</Warning>

<Steps>
  <Step title="停車・安全確保">
    直ちに停車し、二次事故を防ぐため安全な場所へ移動する。
  </Step>
  
  <Step title="負傷者の確認">
    乗客・歩行者の負傷状況を確認する。
  </Step>
  
  <Step title="救急要請">
    負傷者がいる場合は、迷わず**119番**に連絡する。
  </Step>
  
  <Step title="責任者への報告">
    輪タク責任者（渉外担当）に電話で報告する。
  </Step>
  
  <Step title="記録">
    事故の状況、日時、場所、関係者を記録する。写真も撮影する。
  </Step>
</Steps>

<Note>
  軽微な事故でも必ず責任者に報告してください。後日問題になることがあります。
</Note>
