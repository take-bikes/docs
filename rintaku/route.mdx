# 🗺️ 運行に関するまとめ

<Info>
  輪タクは、大学構内の指定されたルートのみを運行します。
  安全確保のため、ルートからの逸脱は禁止です。
</Info>

## 運行ルート概要

1周 約600m / 所要時間 約5分

**ルート**: 南門（スタート地点） → 正門（ゴール地点）, 正門（スタート地点） → 南門（ゴール地点）


![運行ルートマップ](/images/map.png)



## 危険箇所

<Warning>
  以下の箇所は特に注意が必要です。
</Warning>

| 場所 | 危険内容 | 対策 |
|------|----------|------|
| 正門付近 | 車両の出入り | 一時停止、左右確認 |
| 図書館前の坂 | 下り坂 | **最徐行**（時速5km以下） |
| 学食前交差点 | 歩行者多数 | ベル使用、声かけ |

## 速度制限

<Check>
  以下の速度制限を守ってください。
</Check>

| 区間 | 最高速度 |
|------|----------|
| 直線 | 時速10km |
| カーブ・交差点 | 時速5km |
| 下り坂 | 時速5km以下 |
| 人混み | 即停止できる速度 |
